# 🎬 MovieFlakes

A modern movie browsing web app built with **HTML, CSS, and JavaScript** using the **TMDB API**.

MovieFlakes allows users to explore trending movies, popular content, search movies, manage a personal watchlist, and browse categories with an interactive UI.

## 🚀 Live Demo

👉 **GitHub Pages:**  
https://ashifzeeya097-webd.github.io/MovieFlakes/

---

## ✨ Features

### 🎥 Movie Discovery
- Trending movies (Today / This Week)
- What's Popular:
  - Streaming
  - On TV
  - In Theaters
- Movie cards with:
  - Poster
  - Title
  - Genre
  - Release year
  - Rating

### 🔍 Search
- Real-time movie search
- Search results displayed dynamically
- Smooth browsing experience

### ❤️ Watchlist
- Like movies and save them locally
- Persistent watchlist using `localStorage`
- Remove movies anytime

### 📚 Browse Categories
- Navbar-based browsing
- Genre filtering
- Infinite scrolling for large movie collections

### 📱 Responsive Design
- Desktop navigation
- Mobile hamburger menu
- Adaptive segmented controls
- Responsive movie grids and carousels

## 🌐 Deployment

This project is deployed using **GitHub Pages**.

To enable:

1. Go to your repository
2. Open:
   ```
   Settings → Pages
   ```
3. Select:
   ```
   Deploy from branch
   ```
4. Choose:
   ```
   main → /root
   ```
5. Save

Your site will be available at:

```
https://ashifzeeya097-webd.github.io/MovieFlakes/
```
