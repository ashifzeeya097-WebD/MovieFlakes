# Introduction

MovieFlakes is a responsive movie discovery web application built with
vanilla HTML, CSS, and JavaScript using the TMDB API.

## Features

-   🎬 Browse popular and trending movies
-   🔎 Debounced search (500 ms delay after 3+ characters)
-   ❤️ Like and persist favorite movies with Local Storage
-   🎭 Genre, release year, and ratings
-   📱 Responsive desktop and mobile navigation
-   🍔 Mobile hamburger menu
-   🎞️ Smooth horizontal carousels
-   ⏳ Loading animation while fetching data

## Tech Stack

-   HTML5
-   CSS3
-   JavaScript (ES6)
-   TMDB API
-   Local Storage

## Getting Started

1.  Clone the repository.
2.  Open `script.js`.
3.  Replace the TMDB API key with your own if needed.
4.  Open `index.html` in your browser.
